# Análisis de Vehículos - Dashboard Interactivo

## Descripción del Proyecto
Esta aplicación web interactiva permite explorar y analizar datos de vehículos usados mediante visualizaciones dinámicas. Desarrollada con Streamlit y Plotly Express como parte del Sprint 7.

## 🚀 Aplicación en Vivo
**Ver la aplicación desplegada:** https://my-new-repo-1-nk87.onrender.com

## 📊 Funcionalidades
- Histogramas interactivos de datos de vehículos
- Gráficos de dispersión para análisis de correlaciones
- Filtros dinámicos para explorar diferentes segmentos de datos
- Interfaz intuitiva y responsiva

## 🛠️ Tecnologías Utilizadas
- **Python 3.x**
- **Streamlit** - Framework para aplicaciones web
- **Plotly Express** - Visualizaciones interactivas
- **Pandas** - Manipulación de datos
- **Render** - Despliegue en la nube
